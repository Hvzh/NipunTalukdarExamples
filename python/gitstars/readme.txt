This is a script to list the public repositories created by an user.
This may get an idea about how practically active an user may be.


Usage: python gituserrepos.py userid
For example: python gituserrepos.py mattn
****************************************************************

This script list out the public repositories created by an user

IMPORTANT!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
We need to install pygithub3 package
  $ pip install pygithub3
